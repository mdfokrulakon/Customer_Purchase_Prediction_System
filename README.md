The notebook is a simple machine learning project called **“Customer Purchase Prediction System.”**

### Short Summary

* The project is built using **Python** in **Jupyter Notebook**.
* It uses a built-in dataset from **Scikit-learn** (Wine dataset).
* Main goal: train a machine learning model to predict categories/classes from data.

### What the notebook does

1. Imports Python libraries

   * Pandas
   * NumPy
   * Matplotlib
   * Seaborn
   * Scikit-learn

2. Loads the dataset

   * Converts it into a dataframe.

3. Analyzes the data

   * Checks dataset info
   * Statistical summary
   * Missing values

4. Visualizes the data

   * Heatmap
   * Graphs and plots

5. Prepares data

   * Splits into training and testing sets
   * Scales features using `StandardScaler`

6. Trains a machine learning model

   * Uses **Logistic Regression**

7. Makes predictions

   * Predicts target classes on test data.

8. Evaluates performance

   * Accuracy score
   * Confusion matrix
   * Classification report

### Overall

This notebook demonstrates a complete beginner-friendly machine learning pipeline:
**Data Analysis → Data Preprocessing → Model Training → Prediction → Evaluation**.
